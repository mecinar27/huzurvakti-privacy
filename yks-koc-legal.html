2. Toplanan Veriler

2.1. Sınav ve Çalışma Verileri

Konu takibi, deneme sınav sonuçları, çalışma programı ve ilerleme kayıtları yalnızca cihazınızda yerel olarak saklanır. Bu verilere sunuculardan erişilemez.

2.2. OCR ve Yapay Zeka API Kullanımı

Soru fotoğrafı çekimi ve OCR analizi ile AI koçluk raporu oluşturma özelliklerinde Google Gemini API kullanılmaktadır. Bu işlemler sırasında görüntü ve metin verileriniz Google'ın sunucularına iletilir. Gemini API gizlilik politikası: policies.google.com/privacy

2.3. API Anahtarı

Gemini API kullanımı için girdiğiniz API anahtarı yalnızca cihazınızda şifreli olarak saklanır, sunuculara gönderilmez ve üçüncü taraflarla paylaşılmaz.

2.4. Kullanıcı Hesabı

YKS Koç hesap oluşturma veya kayıt gerektirmez. E-posta, isim, kimlik bilgisi toplanmaz.

3. Veri Paylaşımı

Kişisel verileriniz hiçbir üçüncü tarafla paylaşılmaz, satılmaz veya kiralanmaz. Gemini API'ye gönderilen içerikler yalnızca AI işleme amaçlıdır ve Google'ın veri işleme politikasına tabidir.

4. Reklam Sistemi

YKS Koç, ödüllü reklam (rewarded ad) modeli kullanmaktadır. Reklamlar yalnızca kullanıcının isteğiyle başlatılır; zorunlu reklam gösterimi yapılmaz.

  • Belirli özellikler veya ek haklar için isteğe bağlı reklam izlenebilir.
  • Reklamlar Google AdMob altyapısı üzerinden sunulmaktadır.
  • AdMob, reklamları kişiselleştirmek amacıyla reklam kimliği (Advertising ID) kullanabilir.
  • Reklam tercihlerinizi Android cihazınızın Ayarlar → Google → Reklamlar bölümünden yönetebilirsiniz.

5. Veri Paylaşımı

Kişisel verileriniz hiçbir üçüncü tarafla ticari amaçla paylaşılmaz. Gemini API ve AdMob servisleri yalnızca uygulama işlevleri için kullanılır.

6. Veri Güvenliği

Tüm yerel veriler Android'in güvenli depolama alanında saklanır. API çağrıları HTTPS/TLS ile şifrelenerek iletilir.

7. Haklarınız

Uygulama verilerinizi Ayarlar → Uygulama Yönetimi → YKS Koç → Veri Temizle yoluyla silebilirsiniz. KVKK talepleriniz için mecinar27@gmail.com adresine başvurabilirsiniz. Başvurular 30 gün içinde yanıtlanır.

1. Taraflar ve Kapsam

Bu kullanım şartları, YKS Koç uygulamasını sunan mecinar27 (Geliştirici) ile uygulamayı kullanan kişi (Kullanıcı) arasında geçerlidir. Uygulamayı indirerek bu şartları kabul etmiş sayılırsınız.

2. Uygulamanın Amacı

YKS Koç; YKS sınavına hazırlanan öğrencilere konu takibi, deneme analizi, AI destekli koçluk raporu ve çalışma planı özellikleri sunan ücretsiz bir yardımcı araçtır.

3. Yapay Zeka ve Sınav Sonuçları

⚠️ YKS Koç tarafından üretilen AI raporları, puan tahminleri ve koçluk önerileri yalnızca yardımcı bilgi niteliğindedir. Kesin sınav sonuçlarını garanti etmez. Uygulama sonuçlarına dayanarak alınan kararların sorumluluğu kullanıcıya aittir.

4. Gemini API Kullanımı

Uygulamadaki AI özellikleri için kullanıcının kendi Google Gemini API anahtarını girmesi gerekebilir. Bu anahtarın güvenliği kullanıcının sorumluluğundadır. API kullanım ücretleri doğrudan Google tarafından tahsil edilir; geliştirici bu konuda sorumluluk kabul etmez.

5. Kullanım Koşulları

  • Uygulama yalnızca kişisel ve eğitim amaçlı kullanılabilir.
  • Ticari lisans olmaksızın içerik kopyalanamaz veya dağıtılamaz.
  • Uygulamanın tersine mühendislik ile çözülmesi yasaktır.

6. Sorumluluk Sınırları

Uygulama "olduğu gibi" sunulmaktadır. OCR hataları, API kesintileri veya yanlış puan tahminlerinden doğan zararlardan geliştirici sorumlu tutulamaz.

7. Fikri Mülkiyet

📡 Takip Teknolojileri Politikası

▾
Son Güncelleme: 23 Nisan 2026

Kullanılan Teknolojiler

YKS Koç, ödüllü reklam sistemi için Google AdMob SDK kullanmaktadır. Analitik izleme veya kullanıcı profilleme amacıyla başka bir SDK kullanılmaz.

Harici Servis 1: Google AdMob

Ödüllü reklamların gösterimi için kullanılır. AdMob, reklam kişiselleştirmesi amacıyla cihazın Advertising ID'sini kullanabilir. Reklamlar yalnızca kullanıcının isteğiyle başlatılır. Tercih yönetimi: Ayarlar → Google → Reklamlar.

Harici Servis 2: Google Gemini API

OCR ve AI koçluk özellikleri için Gemini API kullanılır. API çağrıları sırasında görüntü ve metin içerikleri Google sunucularına iletilir. İşleme süreci Google'ın gizlilik politikasına tabidir.

Yerel Depolama

Tüm sınav verileri, konu takibi ve ayarlar Android'in yerel SharedPreferences ve Room veritabanında tutulur. Bulut senkronizasyonu yapılmaz.

Kamera ve Galeri

OCR özelliği için kamera ve galeri izni istenir. Çekilen fotoğraflar yalnızca metin çıkarma (OCR) işlemi için Gemini API'ye iletilir; cihazda veya sunucularda ayrıca saklanmaz.

Kontrol ve Tercihler

  • Kamera izni: Cihaz Ayarları → Uygulama İzinleri → YKS Koç
  • Depolama izni: Cihaz Ayarları → Uygulama İzinleri → YKS Koç
  • Tüm uygulama verisi: Ayarlar → Uygulama Yönetimi → Veri Temizle

Risk Değerlendirmesi

YKS Koç sunucu tabanlı kullanıcı verisi barındırmadığından, geleneksel anlamda sunucu taraflı veri ihlali riski yoktur. Kullanıcı verileri tamamen cihazda yerel olarak saklanır.
